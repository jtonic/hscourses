# Haskell Courses

## Prerequisites

- The GHCup Haskell installer, version v0.1.18.0

- Toolchain

```json
"haskell.toolchain": {
    "hls": "1.9.0.0",
    "cabal": "latest",
    "ghc": "9.2.5",
    "stack": "latest"
},
```


## cabal basic commands

- cabal build

- cabal run

- cabal test

- cabal test --test-show-details=direct
